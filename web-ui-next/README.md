## Next.js Frontend (Design Direction)

This folder contains a Next.js implementation inspired by `design/Picture1.png`:

- Editorial hero style ("What are you craving today?")
- Step-based preference builder
- Locality dropdown from backend `/localities`
- Max budget + minimum rating controls
- Cuisine chips
- Recommendation result cards

### Run

1. Copy environment file:
   - `.env.local.example` -> `.env.local`
2. Ensure backend is running on `http://127.0.0.1:8000` (or update env).
3. Install deps and run:
   - `npm install`
   - `npm run dev`

Open: `http://127.0.0.1:3000`

### Note

This is intentionally a base visual direction in Next.js so the UI can be iterated quickly in upcoming phases.
